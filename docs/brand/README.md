# Keepla — Manual de Marca (Fonte Única de Verdade)

## Essência da marca

- Keepla é memória e silêncio. Um guardião do tempo que preserva emoções com cuidado, simplicidade e respeito.
- Tom: emocional, humano, contido — sempre discreto; sem exuberância comercial.

## Paleta cromática (HEX)

- Preto: #000000
- Branco: #FFFFFF
- Cinzentos: #F5F5F5, #E0E0E0, #6B6B6B, #262626
- Vermelho (único acento): #E63946  (USAR EXACTAMENTE ESTE HEX)

## Regras estritas de uso do vermelho

- O vermelho `#E63946` é o ÚNICO acento. Usar apenas para micro-accentos: CTAs principais, links de destaque, indicadores de estado emocional.
- Nunca usar vermelho como textura, gradiente, brilho ou efeito metálico.
- Evitar usar vermelho como fundo de grandes áreas; preferir bordas, ícones, pequenos botões.

## Tipografia

- Preferir tipografia neutra e sem floreados: `Inter` (ou sistema sans-serif equivalente) para UI e textos corridos.
- Frases emotivas ou citadas podem usar uma tipografia serif discreta, mas não obrigatória; manter legibilidade e sobriedade.
- Evitar fontes ornamentais ou display.

## Fotografia

- Fotografias: exclusivamente preto e branco (grayscale 100%).
- Estilo documental: retratos, detalhes, mãos, papel, escrita. Sem filtros estilizados, sem saturação.
- Tratamento: leve aumento de contraste se necessário; evitar granulações artificiais e efeitos metálicos.

## O que NÃO é Keepla

- Proibido: dourado, gold, metálicos, brilho, gradientes, efeitos de luxo/premium, texturas metálicas.
- Proibido: estética corporativa, institucional, tecnológica-futurista, ou ostentação visual.
- Proibido: múltiplas paletas, cores de acento adicionais, ícones coloridos além do vermelho único.

## Princípios práticos

- Se algo não tiver função clara → eliminar.
- Prefira espaços em branco e composição silenciosa.
- Ícones monocromáticos (preto ou vermelho para destaque único).

## Implementação técnica

- Variáveis CSS centrais:
  - `--keepla-red: #E63946`
  - `--keepla-black: #000000`
  - `--keepla-white: #FFFFFF`
  - `--keepla-gray-100: #F5F5F5`
  - `--keepla-gray-200: #E0E0E0`
  - `--keepla-gray-500: #6B6B6B`
  - `--keepla-gray-800: #262626`

- Proibir gradientes e classes utilitárias que criem brilho ou metálicos.
